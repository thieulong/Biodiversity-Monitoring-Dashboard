Biodiversity Monitoring Dashboard

Abstract:
The Biodiversity Monitoring Dashboard is a portable, interactive tool designed to help researchers and practitioners analyze biodiversity survey data. Upload your detection results, visualize trends, and explore detailed statistics—all securely on your local machine.

Technology Stack & Compatibility

  • Stack: Grafana (visualization), Node-RED (data processing), TimescaleDB/PostgreSQL, Docker Desktop.
  • Supported OS: Windows 10/11, MacOS (Intel & Apple Silicon).
  • Requirements: Docker Desktop installed. No internet connection needed after first download.

Setup & Usage Instructions

  1. Unzip the downloaded folder anywhere (e.g., Desktop).
  2. Install Docker Desktop if you haven’t already:
    https://www.docker.com/products/docker-desktop/
  3. Start Docker Desktop (wait until it finishes loading).
  4. Double-click Windows.bat (for Windows) or MacOS.command (for Mac).
  5. Your dashboard will open in a browser. Use the links for:
     • Data upload (Node-RED UI)
     • Results visualization (Grafana)
  6. Grafana login: admin / 0m{-}>7nP8)C
  7. To stop: simply close your browser window.
Note:
On first launch, Windows or Mac may warn you about running these scripts.
• Windows: Click ‘Run anyway’ if prompted.
• Mac: Click ‘Open’, or ‘Open Anyway’ in Security settings if needed.
These warnings are expected and safe. The script only launches the dashboard on your computer.

Troubleshooting & FAQ

Q: Docker Desktop won't start?
A: Make sure virtualization is enabled in your BIOS (Windows), or check Docker’s troubleshooting docs.
Q: Dashboard doesn’t open in browser?
A: Open index.html manually from the unzipped folder, or try http://localhost:1880/ui/ (CSV upload) and http://localhost:3000/ (Grafana dashboard).
